https://surzhik.github.io/team-status/


## Download and Install

Let's go!

```
$ yarn install
```
## Configure
Copy .env.sample to .env and set link to mongodb database
You need to crate collections:
- managers 
- projects 
- skills 
- users 
- workers 

## Development

Run development page on **localhost:3000**

```
$ yarn start
```

## Build

Build for production.

```
$ yarn build
```
